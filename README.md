## Running & Deployment
Serve with hot reload at localhost:3000 for development purposes
```bash
yarn dev
```
Windows:
```bash
yarn win-dev
```

Build for production and launch server
```bash
yarn build
yarn start
```

```bash
# generate static project
$ yarn generate
```
